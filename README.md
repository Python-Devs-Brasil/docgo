docgo: a small tradutor for python's docstring
===============================================

``docgo`` uses goslate to make translations of Python's Docstring.
See goslate for more details.

Usage
=====
```
>>> from docgo import *
>>> dhelp(dir)
 dir ([objeto]) -> lista de strings
 ...
 ```

Install
=======

 ```
 $ pip install docgo
 ```
